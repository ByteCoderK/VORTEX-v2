##  Getting Started

1. Clone the Repository  
   ```bash
   git clone 
   https://github.com/yourusername/VORTEX.git
   ```

2. Install Dependencies  
   ```bash
   pip install -r requirements.txt
   ```

3. Launch VORTEX  
   ```bash
   python VORTEX.py
   ```

4. Speak your first command — try:  
   _"Hey Vortex, what's the time?"_
